# Plan: Convertir stepper-divider en acordeón para formulario de cliente

## Objetivo
Convertir el divisor "o registra uno nuevo" en Paso 2 del cotizador en un acordeón colapsable que muestre/oculte el formulario de registro de cliente.

## Implementación

### 1. Modificar HTML (inventory/index.html)
- Reemplazar `<div class="stepper-divider">` con un botón acordeón
- Agregar icono chevron para indicar estado de expansión
- El botón debe tener clase `stepper-accordion-toggle`
- El formulario debe tener clase `stepper-inline-form` y atributo `data-expanded`

### 2. Agregar estilos CSS (inventory/inventory.css)
- Crear estilos para `.stepper-accordion-toggle`:
  - Apariencia de botón con fondo hover
  - Icono chevron que rota al expandir
  - Transiciones suaves
- Crear estilos para `.stepper-inline-form` con animación de altura:
  - Altura 0 y overflow hidden cuando `data-expanded="false"`
  - Altura auto cuando `data-expanded="true"`

### 3. Agregar lógica JavaScript (inventory/js/screens/quotation-screen.js)
- Agregar evento click al botón acordeón para toggle
- Inicializar el acordeón colapsado por defecto
- Limpiar formulario cuando se colapsa

## Detalles técnicos
- Uso exclusivo de variables CSS existentes (--color-accent, --border-color, etc.)
- Mantener consistencia con el sistema de diseño BEM existente
- Transición de altura con `max-height` para animación suave
