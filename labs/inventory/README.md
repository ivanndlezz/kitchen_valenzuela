# Inventory Labs

Use this folder for unfinished features that are intended for the main `inventory/` app.

Suggested layout:

- `components/` - Isolated component experiments.
- `screens/` - Full-screen or route-level experiments.
- `workflows/` - Multi-step feature experiments.
- `data/` - Sample data used by lab prototypes.

Keep lab code easy to delete, rename, or promote. Once a feature is stable, move the production version into `inventory/js/`, `inventory/css/`, or another app-specific folder.
